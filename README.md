## 🚀 快速开始

### 1. 安装 OpenClaw

```bash
# Windows
iwr -useb https://openclaw.ai/install.ps1 | iex

# macOS/Linux
curl -fsSL https://openclaw.ai/install.sh | bash
```

### 2. 配置技能

```bash
# 安装技能
openclaw skill install <skill-name>

# 查看已安装技能
openclaw skill list
```

### 3. 使用技能

复制本仓库中的技能文档，按照说明配置即可使用。
